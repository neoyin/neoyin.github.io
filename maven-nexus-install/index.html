Nexusжҗӯе»әMavenз§ҒжңҚ(дёҖ) Mavenе®үиЈ…

Mar 21 2012

Mavenе®үиЈ…

  • дёӢиҪҪең°еқҖпјҡhttp://maven.apache.org/download.html
  • дёӢиҪҪ(жӯӨзүҲжң¬дёә3.0.4)пјҡwget http://mirror.bjtu.edu.cn/apache/maven/binaries/apache-maven-3.0.4-bin.tar.gz
  • и§ЈеҺӢе®үиЈ…пјҡtar -xvzf apache-maven-3.0.4-bin.tar.gz
    -mavenзҡ„зҺҜеўғй…ҚзҪ®: vim /etc/profile дёӯ ж·»еҠ 

Nexusе®үиЈ…

  • Nexus зҪ‘з«ҷ http://www.sonatype.org/nexus/
  • дёӢиҪҪ http://nexus.sonatype.org/downloads/

NexusжҸҗдҫӣдәҶдёӨз§Қе®үиЈ…ж–№ејҸпјҢдёҖз§ҚжҳҜеҶ…еөҢJettyзҡ„bundleпјҢеҸӘиҰҒдҪ жңүJREе°ұиғҪзӣҙжҺҘиҝҗиЎҢгҖӮз¬¬дәҢз§Қж–№ејҸжҳҜWARпјҢдҪ еҸӘйЎ»з®ҖеҚ•зҡ„е°Ҷе…¶еҸ‘еёғеҲ°webе®№еҷЁдёӯеҚіеҸҜдҪҝз”Ё.

  • и§ЈеҺӢ tar xvzf nexus-2.0-bundle.tar.gz

иҝӣе…Ҙ cd вҖҰbin/jsw/linux-x86-64/дёӯ иҝҗиЎҢ ./nexus start еҚіеҸҜгҖӮ

http://localhost:8081/nexus иҝӣе…Ҙ. й»ҳи®Өз”ЁжҲ·еҗҚеҜҶз Ғдёәadmin/admin123.

nexusй»ҳи®ӨжҳҜе…ій—ӯиҝңзЁӢзҙўеј•дёӢиҪҪеҠҹиғҪзҡ„пјҢдё»иҰҒжҳҜжӢ…еҝғдјҡйҖ жҲҗеҜ№жңҚеҠЎеҷЁзҡ„е·ЁеӨ§иҙҹжӢ…пјҢйңҖиҰҒжҲ‘д»¬жүӢе·ҘејҖеҗҜгҖӮ

ејҖеҗҜзҡ„ж–№ејҸпјҡ

зӮ№еҮ»AdministrationиҸңеҚ•дёӢйқўзҡ„RepositoriesпјҢе°ҶиҝҷдёүдёӘд»“еә“Apache SnapshotsпјҢCodehaus SnapshotsпјҢMaven Centralзҡ„Download Remote Indexesдҝ®ж”№дёәtrueгҖӮз„¶еҗҺеңЁиҝҷдёүдёӘд»“еә“дёҠеҲҶеҲ«еҸій”®пјҢйҖүжӢ©Re-indexпјҢиҝҷж ·Nexusе°ұдјҡеҺ»дёӢиҪҪиҝңзЁӢзҡ„зҙўеј•ж–Үд»¶гҖӮ

иҰҒи®©mavenдҪҝз”ЁnexusдҪңдёәз§ҒжңҚпјҢйңҖиҰҒеҒҡдёҖдәӣи®ҫзҪ®пјҢдҪҝз”Ёе’ҢеҺҹжқҘи®ҫзҪ®artifactoryзӣёдјјзҡ„ж–№жі•гҖӮдҝ®ж”№~/.m2/settings.xml.

еўһеҠ nexusзҡ„profileпјҡ

й•ңеғҸ:

еҰӮжһңдҪ зҡ„ең°зҗҶдҪҚзҪ®йҷ„иҝ‘жңүдёҖдёӘйҖҹеәҰжӣҙеҝ«зҡ„centralй•ңеғҸпјҢжҲ–иҖ…дҪ жғіиҰҶзӣ–centralд»“еә“й…ҚзҪ®пјҢжҲ–иҖ…дҪ жғідёәжүҖжңүPOMдҪҝз”Ёе”ҜдёҖзҡ„дёҖдёӘиҝңзЁӢд»“еә“пјҲиҝҷдёӘиҝңзЁӢд»“еә“д»ЈзҗҶзҡ„жүҖжңүеҝ…иҰҒзҡ„е…¶е®ғд»“еә“пјүпјҢдҪ еҸҜд»ҘдҪҝз”Ёsettings.xmlдёӯзҡ„mirrorй…ҚзҪ®гҖӮ

жӯӨй…ҚзҪ®з”Ёпјӯavenз§ҒжңҚиҰҶзӣ–centralд»“еә“й…ҚзҪ® иҝҷж ·пјӯavenеҜ№д»»дҪ•д»“еә“зҡ„жһ„д»¶дёӢиҪҪиҜ·жұӮйғҪдјҡиҪ¬еҲ°з§ҒжңҚдёӯ.

йңҖиҰҒејҖжңәеҗҜеҠЁmavenз§ҒжңҚжңҚеҠЎ

  • зӣёе…іең°еқҖ: http://maven.apache.org/pom.html#Quick_Overview
