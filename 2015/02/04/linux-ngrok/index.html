    • ngrok-神器让你外网访问本地

    ngrok 是一个反向代理，通过在公共的端点和本地运行的 Web 服务器之间建立一个安全的通道。ngrok 可捕获和分析所有通道上的流量，便于后期分析和重放

    神器，ngrok绝对是一件神器
    来看一下他的使用帮助

    执行ngrok -proto=tcp 22进行tcp端口转发就可以在外网ssh到本机了

    ngrok利用随机端口进行转发.
    你可以用ssh username@ngrok.com -p 45629可以外网登陆了

    可以采用自定义二级域名test.ngrok.com转发本机的8080端口。
