    最近研究一下TensorFlow 想升级一下mac下的python

    网上找到脚本如下

    mac下/System/Library/Frameworks目录没有权限
    解决方法:
    重启电脑,长按Command+r 进入Recovery模式
    打开终端
    执行如下命令:
    csrutil disable
    重启即可.
