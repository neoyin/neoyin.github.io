1.4 Windows 系统

  1. 从 Nginx官方网站 下载Windows版本
  2. 解压到指定目录，如 C:\nginx
  3. 打开命令提示符，进入Nginx目录
  4. 运行 nginx.exe 启动服务

2. 基本配置

2.1 配置文件结构

Nginx的主要配置文件位于：

  • Ubuntu/Debian: /etc/nginx/nginx.conf
  • CentOS/RHEL: /etc/nginx/nginx.conf
  • 编译安装: /usr/local/nginx/conf/nginx.conf
  • macOS (Homebrew): /usr/local/etc/nginx/nginx.conf
